首先声明“向量”（Vector）

点积

几何意义是两个向量的长度之积乘它们夹角的余弦值。
设求$ \vec a = ( x_1, y_1 ) $, $ \vec b = ( x_2, y_2 ) $两向量的点积。
则$ \vec a \cdot \vec b = x_1 x_2 + y_1 y_2 $
具体如下：
令$\alpha$是$\vec a$与$x$轴的夹角，$\beta$是$\vec b$与$x$轴的夹角。（任意角）
$$
\vec a \cdot \vec b
= |\vec a| \cdot |\vec b| \cdot cos(\alpha-\beta)
= |\vec a| \cdot |\vec b| \cdot ( cos\alpha \cdot cos\beta + sin\alpha \cdot sin\beta )
= x_1x_2 + y_1y_2
$$
那么，两个向量的夹角我们就可以通过arccos算出来了（c++里是写作acos()）。

叉积

这里所说的叉积 是代表两个向量组成的有向三角形的两倍。
由于三角形面积是$ S_\Delta = \frac{ 水平宽 \times 铅锤高 }{2} $
所以很好列出叉积计算公式（注意是有向的）
$$
\vec a \times \vec b
= x_2 \cdot ( x_1 \cdot \frac{y_2}{x_2} - y_1 )
= x_1y_2 - x_2y_1
$$

叉积有一些不同之处

  • $ \vec a \times \vec b = - \vec a \times \vec b $
  • $ \vec a \times \vec a = 0 $
  • $ \vec a \times \vec 0 = 0 $
  • $ \vec a \times ( \vec b + \vec c ) = \vec a \times \vec b + \vec a \times \vec c $
  • $ ( \vec a + \vec b ) \times \vec c = \vec a \times \vec c + \vec b \times \vec c $
  • $ ( \lambda \vec a ) \times \vec b = \lambda ( \vec a \times \vec b ) = \vec a \times ( \lambda \vec b ) $
  • $ \vec a \times \vec b + \vec c \times \vec d = ( \vec a - \vec c ) \times ( \vec b - \vec d ) + \vec a \times \vec d + \vec c \times \vec b $ 注意叉积满足的是反交换律

两个向量的位置关系

不妨设$ u = \vec a \cdot \vec b $, $ v = \vec a \times \vec b $ .
那么，将$\vec a$作为新坐标系的y轴，x轴垂直指向$\vec a$的右侧。在这个新坐标系中点(u,v)所在象限就刻画了$\vec a,\vec b$的相对位置。（即$\vec b$指向$\vec a$的左前还是左后还是右前还是右后）。

向量旋转任意角

设$\vec a = ( x_1, y_1 )$旋转$\theta$到$\vec b = ( x_2, y_2 )$
设$\alpha$是$\vec a$与x轴夹角（任意角）。
$$
x_2
= |\vec a| \cdot cos ( \theta + \alpha )
= |\vec a| \cdot ( cos\theta \cdot cos\alpha - sin\theta \cdot sin\alpha )
= |\vec a| \cdot ( \frac{x_1}{|\vec a|} - \frac{y_1}{|\vec a|} )
= x_1 \cdot cos\theta - y_1 \cdot sin\theta
$$

$$ y_2 = |\vec a| \cdot sin ( \alpha + \theta ) = |\vec a| \cdot ( sin \alpha \cdot cos \theta + cos \alpha \cdot sin \theta ) = |\vec a| \cdot ( \frac{y_1 \cdot cos \theta }{ |\vec a| } + \frac{x_1 \cdot sin \theta }{|\vec a|} ) = x_1 \cdot sin \theta + y_1 \cdot cos \theta $$
所以，旋转后的向量是$ \vec b = ( x_1 \cdot cos \theta - y_1 \cdot sin \theta, x_1 \cdot sin \theta + y_1 \cdot cos \theta ) $

单位法线

向量的法线是通过逆时针旋转$90^。$得到的。
单位法线是指长度为单位长度。
